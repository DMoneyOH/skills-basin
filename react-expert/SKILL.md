# react-expert
React expert is a set of guidelines for building production-ready React applications with optimal performance. It covers various aspects of React development, including component design, hook patterns, state management, and performance optimization.
## When to Use / When NOT to Use
Use React expert when writing, reviewing, or refactoring React/Next.js code to ensure optimal performance. Do not use it for simple static websites or applications that do not require complex state management or performance optimization.
## Core Workflow or Key Techniques
The core workflow of React expert involves designing components, using hooks, managing state, and optimizing performance. Key techniques include:
* Designing components with a single responsibility
* Using hooks for state management and side effects
* Managing state with useState, useReducer, and Context
* Optimizing performance with memoization, virtualization, and profiling
## Quick Reference
* Use useState for simple state management
* Use useReducer for complex state management
* Use Context for shared state management
* Use memoization and virtualization for performance optimization
